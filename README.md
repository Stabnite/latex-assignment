# latex-assignment

You will have to paste the book into packages/book.py

1. Get a PDF copy of `Discrete Mathematics and Its Applications 8th Edition'
2. Open PDF and copy all (CTRL + A then CTRL + C)
3. Paste into packages/book.py between the quotes (''')

# usage

1. Run main
2. Copy exercises (make sure you have the keyword `Exercises` in your selection)
3. Paste latex
